# mtd-api-frontend

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development

#### Frontend

```
yarn run serve
```

#### Backend
```
node server.js
OR
nodemon server.js (for hot reloading)
```

### Note

Currently only contains Self Assessment APIs. Needs UI redesign as currently pretty ugly. Add new APIs to `src/data.js`.
